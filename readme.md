## Installation

`pipx install "git+https://github.com/skarfie123/PythonImageViewer.git"`

## Usage

### View some images

`python-image-viewer *.jpg *.png`

### Change display mode (height, stretch, width)

`python-image-viewer *.jpg -m stretch`
